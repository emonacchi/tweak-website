tweak anatomy

Some highlights of our UI
  • 1. Global settings
    • Search on/off: read more about the Search functionality here
    • Delete all: deletes all the configurations and the request autocomplete cache. Retrigger in the browser tab the requests that you wish to search in the URL field.
    • Collapse all rows: minimizes all the rows
    • Run/Pause the extension: click on the Run button when you are ready to start intercepting requests
    • Create a new configuration: adds a new empty configuration as a row
  • 2. Import/Export You can use the export functionality to save configurations and test cases that can be imported back later on
  • 3. Regular expression toggle When toggled on, you can specify a Regular Expression to match against the request URL. Don't worry about typos, we will give you instant feedback on whether your regular expression is valid or not.
  • 4. Number of requests intercepted If you're seeing the 0 value after trying to intercept a request it means that you might have a mismatch on your configuration, double check the request URL and the selected HTTP method to ensure everything is ok.
  • 5. Row Options Here you have the additional options of duplicating or deleting a configuration.
  • 6. Delay request by Allows you to delay the request interception by as many milliseconds (ms) as specified.
  • 7. Request payload You can specify here the request payload for each request.

Easily search and complete the requests you want to intercept
  • How do I turn search on?
    • Click on the search icon in the top bar
    • In the browser tab, trigger the requests you want to autocomplete
    • Search for the request directly in the URL field (just like you do in the DevTools - Network panel)
    • Select a request and all the fields will be autocompleted for you üéâ
  • What parameters can I search for?
    • Search by request URL
    • Search by HTTP method: if you search for "GET", you will see all the GET requests available. Note: all the HTTP method keywords are case-sensitive.
  • Can I use regular expressions to search? Yes, if you turn on the regular expression toggle, you will be able to filter results with regular expressions.
