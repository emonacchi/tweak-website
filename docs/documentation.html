Documentation

Follow the steps to intercept your first request with tweak!

Step 1 - Request setup

  • Open the DevTools Network tab
  • In the Name section, select the request you want to intercept if you need to collect information about the request
  • Open tweak and fill in Configuration name, HTTP Method and URL match expression* according to the caracteristics of the request you want to intercept
* the URL match expression can either be pure text that tweak will try to match against the URL with simple string inclusion or in alternative you can use a regular expression, just use the prefix rx:, for example rx: /api/users/1[0-9]+ will intercept all the requests that hit this users endpoint with an id between 10 and 19.

Step 2 - Raw response data

  • Copy the Raw response data from the response section
  • Paste the Raw response data in the json editor in tweak

Step 3 - Format the response

In the json editor, you can format the data by clicking on the first icon. The data will be more readable and you can start tweaking the response as you need.

Step 4 - tweak the response payload

tweak the response in the json editor by adding, removing or editing existing data fields.

Step 5 - Run the extension

Go to the menu on the top left of the extension and click Run.

By default, the extention is paused as it could interfere with the websites that you normally use.

It is a good practice to keep the extention turned off when you are not using it, otherwise it would perform unnecessary work in the background.

Step 6 - Switch on the Request

Switch on the request(s) that you want to intercept.

Step 7 - Request intercepted!

Reproduce the event that triggers the data fetching, in this example by clicking on the refresh button.

When the request is intercepted, you will see a number near the switch in the extention. This number displays the amount of requests successfully intercepted.

Do you want to try it yourself in the same application? Here you can access the website shown in these steps.

About tweak

tweak is an extension that allows you to tweak your HTTP requests so that you can test, develop and demo your web application.
